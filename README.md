# 🚀 Full Stack Web Development Setup Guide (Using AI)

Welcome Students 👋
Before starting the Full Stack Web Development workshop using AI tools, please make sure your laptop is properly set up with the required software.

---

## 📌 Mandatory Software Installation

Install the following tools on your laptop:

* Visual Studio Code
* Node.js
* PostgreSQL
* Postman
* Git
* GitHub Desktop
* Google Chrome
* ChatGPT Account (Sign up required)

---

## 🛠️ Installation Links

Use the official links below to download:

* VS Code: https://code.visualstudio.com/
* Node.js: https://nodejs.org/
* PostgreSQL: https://www.postgresql.org/download/
* Postman: https://www.postman.com/downloads/
* Git: https://git-scm.com/downloads
* GitHub Desktop: https://desktop.github.com/
* Google Chrome: https://www.google.com/chrome/
* ChatGPT: https://chat.openai.com/

---

## ⚙️ VS Code Extensions (Must Install)

After installing VS Code, install these extensions:

* Live Server
* Prettier
* ES7 React Snippets
* Thunder Client

### Steps:

1. Open VS Code
2. Click on Extensions (Left sidebar)
3. Search extension name
4. Click Install

---

## ✅ Verification Checklist

Make sure everything is working:

* Node.js installed → Run `node -v` in terminal
* Git installed → Run `git --version`
* VS Code opens properly
* PostgreSQL installed successfully
* Postman opens without errors
* GitHub Desktop logged in
* ChatGPT account created

---

## 💡 Important Notes

* Bring your laptop fully charged 🔋
* Ensure stable internet connection 🌐
* Install all tools before the session
* Create GitHub account if not already done

---

## 🎯 Objective

After setup, you will be able to:

* Build full stack websites
* Use AI (ChatGPT) for development
* Create APIs using Node.js
* Manage databases using PostgreSQL
* Test APIs using Postman
* Upload projects on GitHub
